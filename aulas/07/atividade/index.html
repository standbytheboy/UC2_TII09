Síntese das Propriedades CSS

  1. color

    2. Define a cor do texto de um elemento.

    h1 {
    color: blue;
}
  2. text-align

    3. Alinha o texto dentro de um elemento.

    • left: alinha o texto à esquerda (padrão).
    • center: centraliza o texto.
    • right: alinha o texto à direita.
    h1 {
    text-align: center;
}
  3. font-size

    4. Define o tamanho da fonte de um elemento.

    p {
    font-size: 10px;
}
  4. background-color
    5. Define a cor de fundo de um elemento. 
    button {
    background-color: yellow;
}
  5. color (para botões)

    6. Define a cor do texto de um elemento (usado em conjunto com background-color)

    button{
    color: black;            
}
