HTML

HTML (Linguagem de Marcação de Hipertexto), é o bloco de construção mais básico da Web. Define o significado e assinatura de conteúdo Web.

Tags/Marcações

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador Web.

Um elemento HTML é separado de outro texto em um documento por "tags". As tags consistem no nome do elemento entre um "<" e ">".

Tags Headers

São tags utilizadas para formatar texto. Entre as opções, temos de "h1" até "h6" que alteram tamanho e espessura de fontes.

Tag Meta

Recebe o argumento "charset" que é utilizado para identificar o tipo de caracteres utilizado no código.
